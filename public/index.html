dig +short

Easy-to-use dig API. Equivalent to "dig +short".

How to use it?

http://digshort.net/[domain]/[record].[format]

Supported formats

Currently JSON and plain text are supported.

Examples

$ curl http://digshort.net/example.com/A.json

["192.0.43.10"]
